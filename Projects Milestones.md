# Stock Trend AI Project

## Milestone 1 – Foundation of Stock Prediction System

**Description**

Milestone 1 focused on building the core machine learning pipeline for stock prediction. Historical Tesla stock data from the past five years was collected and preprocessed. Data cleaning included removing invalid rows, formatting dates, and ensuring numeric columns were properly structured. Feature engineering techniques were applied to generate useful features such as lag values and moving averages. A Random Forest Regressor model was then trained to predict the next day's closing stock price. The trained model and scaler were saved for reuse in future predictions.

**Summary**

Milestone 1 established the basic machine learning framework including data collection, preprocessing, feature engineering, and training a baseline Random Forest model for next-day stock price prediction.

---

## Milestone 2 – Model Improvement and Advanced Features

**Description**

Milestone 2 focused on improving the system’s accuracy and intelligence. Missing values were handled and outliers were detected using statistical methods such as IQR and Z-score, with outlier capping applied to maintain realistic market data. Additional technical indicators were introduced including EMA, RSI, MACD, volatility, and daily return percentage. A Long Short-Term Memory (LSTM) deep learning model was implemented to capture long-term patterns in time-series data. The system was also extended to support multi-day forecasting (7–30 days).

**Summary**

Milestone 2 enhanced data quality, introduced advanced technical indicators, and implemented an LSTM deep learning model to support more accurate multi-day stock predictions.

---

## Milestone 3 – Interactive AI Dashboard

**Description**

Milestone 3 focused on developing an interactive Stock Trend AI Dashboard. Users can upload stock datasets in CSV format and preview the data. The dashboard provides visual analytics including data distribution, stock high/low analysis, moving averages, volatility trends, and AI-generated trading signals. Users can also input the number of days for future predictions, and the system generates forecast charts showing potential stock trends. An overall market status summary is displayed to help users interpret stock performance.

**Summary**

Milestone 3 introduced a user-friendly dashboard that enables dataset uploads, stock visualization, analytics, and AI-based future predictions through interactive charts.
